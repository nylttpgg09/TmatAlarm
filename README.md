# TmatAlarm
用于控制自己学习时长，让自己劳逸结合的类似番茄钟的小软件

设置界面：1.设置黑屏时间，不影响其他软件运行，适合摸鱼
2.设置工作模式，学习一定时间进入黑屏状态，无限循环，可通过快捷键退出
